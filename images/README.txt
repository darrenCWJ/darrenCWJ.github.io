Drop your hero images for case studies in this folder.

Recommended specs:
- Format: JPG or PNG
- Size: 1600 x 900px or larger (landscape)
- File size: under 1 MB for fast loading

Naming convention:
  tribunal-hero.jpg
  hometongue-hero.jpg
  hustle-sg-hero.jpg

Then update the src in the case study file:
  <img src="images/tribunal-hero.jpg" ... style="display:none" ...>

Remove the  style="display:none"  to make it show.
The image auto-hides if the file is missing, so the CSS fallback takes over.
